Privacy Policy

OctoService is a service-providing application that allows users to browse available workers by category and contact the service administrator.

Information We Collect

The application only requests a username, which is used to personalize the user experience within the app.

Use of Information

The username is used exclusively for identification and communication with the administrator.

Communication via WhatsApp

When selecting a service, the application redirects the user to WhatsApp, where communication is managed directly by the administrator. OctoService does not store WhatsApp messages.

Data Sharing

No personal information is shared with third parties.
